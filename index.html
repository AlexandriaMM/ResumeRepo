Back at BW, YJ4L!

As I approached graduation from Parma Senior High in 2012, I had my eyes set on Baldwin Wallace University. During high school, I was an active member of the performing arts community and saw BW as the next logical step. Fast forward to my freshmen year at Baldwin Wallace and my goal was to become a music therapist with guitar and vocals being my primary instrument.

So how did my major become computer science?

Prior to BW, I had zero exposure to the field of computer science. My first experience with programming was at an ACM (Association for Computing Machinery) LAN party. Although there were plenty of video and card games to capture my attention, my curiosity peaked the moment I noticed a few students working on a programming assignment. The code on their screen looked entirely different from anything I had seen before and I couldn’t help but ask what they were up to. After these kind students gave me a basic explanation of their assignment, I enrolled in the Intro to Computer Science course offered at Baldwin Wallace and I was not disappointed. Everything I learned in Intro to Computer science only made me want to learn more. In a matter of months, my major had changed to Computer Science and my full attention was dedicated to becoming a software developer.

My initial concerns were as follows: Will I be able to keep up in classes even though I have no background in the field? Am I smart enough for this?

Being a first-generation college student, there were times when I felt as though I didn’t belong. Doubts and insecurities lingered in my mind as my physical and mental health suffered so much that I had to make the tough decision to leave Baldwin Wallace in 2015. In the time period from 2015 until 2023, I moved to another state and started my first job as a software developer. My top priority was to focus on my mental and physical health, meeting with doctors, and receiving the appropriate treatments necessary. I knew that if my mind and body were in a better place, I would have the power to reach all of my personal goals. In 2020, we all know what happened: the Covid-19 pandemic. I made the decision, with my husband, to return back to Ohio so that I could have the ability to visit with my grandmother, Marion Macko, who was battling cancer. In July 2022 my grandmother moved on from this world. My heart was broken and I began seriously contemplating what I want out of this life. I decided that graduating from Baldwin Wallace University with my bachelor's degree in Computer Science would be my number one priority and it still is to this day.

I am overwhelmed with excitement since returning to Baldwin Wallace University as of Spring semester 2023 and look forward to graduation in 2025! YJ4L!

Coursework

Information Technology Concepts for Programmers

Credits: 3         Final Grade: A

Designed for students pursuing careers in programming, networking, database, and comprehensive Information Technology disciplines. Builds a foundation in these concepts through hands-on practice with a variety of programming languages and leading industry tools.

Programming Logic

Credits: 3         Final Grade: A

Learn to solve business problems by designing, coding, and testing programming solutions using a current high-level programming language. Learn and apply standard language constructs, control flow, and beginning object-oriented programming concepts.

Introduction to Web Programming

Credits: 3         Final Grade: A

Build Web pages using current technologies including but not limited to HTML, Cascading Style Sheets, and JavaScript. Focus is on developing a foundation in web programming.

Java Programming

Credits: 4         Final Grade: A

Introduction to object-oriented methodologies and programming using the Java programming language. Design, code, and debug Java applications. Other topics include GUI components, event handling, and exception handling.

Systems Analysis and Design

Credits: 3         Final Grade: A

Overview of systems development life cycle. Utilize structured tools and object-oriented techniques to analyze and document process flow, data flows, data structures, file designs, input & output designs and program specifications in the systems development life cycle. Examine information gathering and reporting activities. Analyze strategies and techniques for producing logical methodologies which deal with complexity in development of information systems.

Data Structures and Algorithms

Credits: 4         Final Grade: A

Programming and problem-solving skills are further developed by using language features to implement various data structures such as stacks, queues, linked lists, trees and graphs. Additional topics include recursion, sorting, searching, and hashing algorithms.

Enterprise Database Systems

Credits: 4         Final Grade: A

Apply knowledge of: relational algebra, data migration, data warehousing, data mining, distributed databases and security to design, develop and normalize a Structured Query Language (SQL) database to 3rd normal form using appropriate diagrams and database objects. Retrieve, insert, update, delete, troubleshoot and report data from complex SQL databases.

Interactive Internet Programming

Credits: 4         Final Grade: A

Introduction to interactive object-oriented programming in an Internet environment from a conceptual approach. Emphasis is on understanding the basic Internet technologies (mostly from the client side), how and when to use them and how to integrate them into a system.

Computer Architecture and Organization

Credits: 3         Final Grade: In Progress

This course covers computing history, computer technology and terminology, basic logic design, computer architecture, CPU architecture, memory systems and assembly language.
